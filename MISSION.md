# Mission

> 中文说明：
> AIWorkbench 的目标不是替代开发者。
>
> 而是打造一套可以持续积累的软件研发平台。

---

## 我们相信

AI 不只是回答问题。

AI 应该成为长期合作的工程伙伴。

---

## 我们希望

减少重复劳动。

沉淀知识。

统一规范。

持续积累。

自动化开发。

---

## 我们不会

为了使用 AI 而使用 AI。

不会引入复杂但没有价值的技术。

不会重复建设已经存在的能力。

---

## 最终目标

开发任何新项目，都只需要一句话：

> "按照 AIWorkbench 标准初始化项目。"

AI 即可完成：

- 创建目录
- 初始化 Git
- 创建 Docker
- 创建数据库
- 创建 API
- 创建文档
- 配置 AI
