# AIWorkbench

> 中文说明：
> AIWorkbench 是一个 AI 驱动的软件研发平台。
>
> 它不是单一项目，而是多个 AI Agent、多个 Workspace、多个业务项目的统一工作平台。

---

## Vision（愿景）

打造一个本地优先（Local First）的 AI 开发平台。

所有 AI（Codex、Qwen3、GPT 等）遵循统一规范，共享知识，共享技能，共享工作流程。

---

## Directory（目录）

```text
ai/
        AI 行为规范、知识、技能、Prompt

shared/
        全局共享工程规范

workspaces/
        所有业务项目
```

---

## Principles（原则）

- Local First
- AI First
- Automation First
- Reusable
- Engineering Driven
