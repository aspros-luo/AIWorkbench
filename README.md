# AIWorkbench

> Workspace Operating System for AI

Version: **2.0.0-alpha**

Codename: **Workspace Brain**

---

## 什么是 AIWorkbench

AIWorkbench 并不是一个 AI CLI。

它也不是一个 Prompt 管理工具。

更不是一个 Agent Framework。

AIWorkbench 是一个 **Workspace Operating System（工作空间操作系统）**。

它让 AI 能够在一个持续成长的 Workspace 中：

- 理解（Understand）
- 协调（Coordinate）
- 执行（Execute）
- 演化（Evolve）

---

## 为什么存在

今天的大多数 AI 工具都有一个共同的问题：

AI 只理解当前上下文。

聊天结束以后：

上下文消失。

知识消失。

经验消失。

工作流消失。

AIWorkbench 希望改变这一点。

Workspace 应该拥有持续成长的能力。

AI 不应该每一次都重新开始。

---

## 核心理念

Workspace 是唯一核心。

Brain 是唯一协调者。

Knowledge 保存事实。

Memory 保存上下文。

Tool 执行能力。

Agent 可替换。

所有能力共同服务于 Workspace。
