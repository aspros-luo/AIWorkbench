# AIWorkbench Product Model

Version: 2.0.0-alpha

Status: Frozen

---

# Product Definition

AIWorkbench is a Digital Employee Operating System.

It is not:

- an AI Chat
- an AI IDE
- an AI Agent Framework

Its purpose is to help users create, train and manage AI Employees.

---

# Core Idea

Each Workspace represents one AI Employee.

Users do not create projects.

Users hire employees.

Every employee continuously learns from:

- conversations
- documents
- screenshots
- workflows
- operation history
- feedback

The employee becomes increasingly capable of working independently.

---

---

# Long-term Goal

An employee should eventually be able to complete an entire business workflow with minimal human intervention.

Examples:

- Temu Operator
- Game Developer
- DevOps Engineer
- Website Manager
- Financial Assistant

---

# Product Philosophy

Train AI Employees instead of using AI Chats.
